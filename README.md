
## Teste técnico front-end WeON

CRUD de endereços seguindo regras recebidas.


### Tecnologias Utilizadas
- Vuejs 2
- Vuex
- Vue Router
- Axios
- Vuetify
- Moment.js
- Vue i18n


### Instalação e Configuração (Comandos)
#### Instalação de dependências:
```
yarn
```
ou
```
npm i
```

#### Execução (localhost:8080)
```
yarn serve
```
ou
```
npm run serve
```
